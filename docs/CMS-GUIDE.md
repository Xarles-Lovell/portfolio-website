## 登录 CMS

1. 访问你的网站 URL，在末尾添加 `/admin`
   - 例如：`https://your-username.github.io/admin`

2. 点击"Login with Netlify Identity"按钮

3. 首次登录需要：
   - 在 Netlify 中配置 Identity 服务
   - 创建管理员账号
   - 详细步骤请参考 [部署指南](./DEPLOYMENT.md)

4. 登录成功后，你会看到 CMS 管理界面

## 编辑个人介绍

1. 在左侧菜单点击"个人介绍"
2. 点击"个人信息"
3. 编辑以下内容：
   - **头像**：点击"选择图片"上传头像（建议尺寸：120x120 像素）
   - **个人介绍**：在文本框中输入介绍文字，支持 Markdown 格式
4. 点击右上角"保存"按钮
5. 等待几分钟，GitHub Actions 会自动构建并部署更新

### Markdown 格式示例

```markdown
你好，我是 **张三**，一名专注于*平面设计*的设计师。

我的设计风格注重：
- 简洁明了
- 视觉冲击力
- 用户体验

欢迎浏览我的作品集！
```

## 管理作品分类

### 添加新分类

1. 在左侧菜单点击"作品分类"
2. 点击右上角"新建作品分类"按钮
3. 填写以下信息：
   - **分类名称**：显示给访客的名称（例如：小说封面）
   - **URL 标识**：用于网址的标识（例如：novel-cover）
     - 只能包含小写字母、数字和连字符
     - 不能包含空格或特殊字符
   - **排序**：数字越小越靠前（例如：1, 2, 3...）
4. 点击"保存"

### 编辑或删除分类

1. 在"作品分类"列表中点击要编辑的分类
2. 修改信息后点击"保存"
3. 或点击"删除"按钮删除分类（注意：删除前请确保没有作品使用该分类）

## 添加和编辑作品

### 添加新作品

1. 在左侧菜单点击"作品"
2. 点击右上角"新建作品"按钮
3. 填写以下信息：
   - **作品标题**：作品的名称
   - **URL 标识**：用于网址的标识（例如：my-first-work）
   - **所属分类**：从下拉菜单选择分类
   - **作品图片**：上传作品图片（建议尺寸：1200x900 像素，4:3 比例）
   - **排序**：在分类中的显示顺序
   - **项目简介**：详细描述作品，支持 Markdown 格式
   - **交互热点**：可选，添加设计说明热点（见下一节）
4. 点击"保存"

### 编辑作品

1. 在"作品"列表中点击要编辑的作品
2. 修改信息后点击"保存"

### 删除作品

1. 在作品编辑页面点击"删除"按钮
2. 确认删除

## 添加交互热点

交互热点可以让访客在作品图片上看到你的设计思路说明。

### 添加热点

1. 在作品编辑页面，找到"交互热点"部分
2. 点击"添加交互热点"按钮
3. 填写以下信息：
   - **热点 ID**：唯一标识（例如：hotspot-1, hotspot-2）
   - **形状**：选择矩形或圆形
   - **位置**：设置热点在图片上的位置
     - **X 坐标 (%)**：距离左边的百分比（0-100）
     - **Y 坐标 (%)**：距离顶部的百分比（0-100）
     - **宽度 (%)**：热点宽度百分比（0-100）
     - **高度 (%)**：热点高度百分比（0-100）
   - **说明文字**：鼠标悬停时显示的设计说明
4. 点击"保存"

### 如何确定热点位置？

1. 在图片编辑软件（如 Photoshop）中打开作品图片
2. 使用标尺或选区工具测量要添加热点的区域
3. 计算百分比：
   - X 坐标 = (距离左边的像素 / 图片总宽度) × 100
   - Y 坐标 = (距离顶部的像素 / 图片总高度) × 100
   - 宽度 = (热点宽度像素 / 图片总宽度) × 100
   - 高度 = (热点高度像素 / 图片总高度) × 100

### 示例

假设图片尺寸为 1200x900 像素，你想在图片中心添加一个 200x150 像素的热点：

- X 坐标 = (500 / 1200) × 100 = 41.7%
- Y 坐标 = (375 / 900) × 100 = 41.7%
- 宽度 = (200 / 1200) × 100 = 16.7%
- 高度 = (150 / 900) × 100 = 16.7%

## 管理联系方式

1. 在左侧菜单点击"联系方式"
2. 点击"联系信息"
3. 在"联系方式列表"中：
   - 点击"添加联系方式列表"添加新的联系方式
   - 填写以下信息：
     - **图标**：选择图标类型（邮箱、微信、微博等）
     - **显示标签**：显示的文字（例如：邮箱）
     - **内容**：邮箱地址、链接或文本
   - 点击"删除"按钮删除不需要的联系方式
4. 点击"保存"

### 联系方式示例

| 图标 | 显示标签 | 内容 |
|------|---------|------|
| email | 邮箱 | your.email@example.com |
| wechat | 微信 | your-wechat-id |
| weibo | 微博 | https://weibo.com/your-profile |
| instagram | Instagram | https://instagram.com/your-profile |

## 网站设置

1. 在左侧菜单点击"网站设置"
2. 点击"通用设置"
3. 编辑以下设置：
   - **网站标题**：显示在浏览器标签的标题
   - **私密模式**：
     - 开启：网站不会被搜索引擎收录，只能通过链接访问
     - 关闭：网站可以被搜索引擎收录
   - **访客统计**：
     - **启用统计**：是否启用 Google Analytics
     - **Google Analytics ID**：你的 GA 跟踪 ID（例如：G-XXXXXXXXXX）
4. 点击"保存"

### 何时使用私密模式？

- ✅ 作品集还在完善中，不想被搜索引擎收录
- ✅ 只想通过简历链接分享给特定人群
- ✅ 测试网站功能

### 何时关闭私密模式？

- ✅ 作品集已经完善，想要更多曝光
- ✅ 希望通过搜索引擎获得访客
- ✅ 正式上线运营

## 常见问题

### Q: 保存后多久能看到更新？

A: 保存后，GitHub Actions 会自动构建和部署网站，通常需要 2-5 分钟。你可以在 GitHub 仓库的 "Actions" 标签页查看部署进度。

### Q: 如何预览更改？

A: Decap CMS 目前不支持实时预览。建议先在本地开发环境测试，或者在 GitHub 上创建一个测试分支。

### Q: 上传的图片太大怎么办？

A: 建议使用图片压缩工具（如 TinyPNG、Squoosh）压缩图片后再上传。推荐尺寸：
- 头像：120x120 像素
- 作品图片：1200x900 像素（4:3 比例）

### Q: 如何备份内容？

A: 所有内容都存储在 GitHub 仓库中，GitHub 会自动保存历史记录。你也可以定期下载仓库到本地作为额外备份。

### Q: 忘记 CMS 密码怎么办？

A: 在 Netlify Identity 管理界面可以重置密码，或者重新邀请自己。

### Q: 可以有多个管理员吗？

A: 可以。在 Netlify Identity 中可以邀请多个用户，但建议只给信任的人管理权限。

### Q: 交互热点在移动设备上如何显示？

A: 在移动设备上，访客需要点击热点区域才能看到说明文字（而不是悬停）。

### Q: 如何删除已上传的图片？

A: 在 GitHub 仓库的 `public/images/uploads` 目录中可以手动删除不需要的图片文件。
